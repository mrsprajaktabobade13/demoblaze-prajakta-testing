# demoblaze-prajakta-testing
## Objective: Design and implement functional tests for the following scenarios of the E-commerce Application (https://www.demoblaze.com/).


## Scope of Testing
The following features were tested:
-User Signup(Positive and Negative)
-User Login(Positive and Negative)
-Product Browsing
-Category Navigation
-Add to cart
-Checkout process
-Logout

##Testing Types
Manual Functional testing
Black Box testing
End to End Scenario testing
